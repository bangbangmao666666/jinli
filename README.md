# 锦鲤 (Jinli) - 高考志愿填报智能顾问

> **守护普通家庭孩子的奋斗成果。**

本项目是一个专为 **2026 年夏季高考**设计的智能志愿填报决策支持系统。我们主要服务于希望通过高考改变命运的普通家庭考生。
使用说明：https://www.bilibili.com/video/BV12WRQBFEdj/?vd_source=719e9a4599dc9708bdd6ddd36128cb2d

## 1. 项目目的与作用

高考是普通家庭孩子最重要、也最公平的向上通道。然而，填报志愿时的信息差和决策失误往往会让孩子数年的努力打折扣。

**锦鲤** 的核心作用是：
- **消除信息差**：通过结构化的院校专业数据库，提供超越官方指南的深度评估。
- **科学决策**：将感性的“我想报”转化为理性的“我能报、值得报、报了稳”。
- **闭环方案**：从原始数据导入到最终的 96 个志愿（以山东为例）排列，提供全流程指导。

## 3. 项目缺陷与局限

本项目仍处于 MVP (最小可行性产品) 阶段，存在以下局限：
- **数据高度依赖**：系统本身不内置全量招生数据，高度依赖用户自行导入的该省份招生计划、一分一段表及历史录取数据。
- **待实践检验**：目前的运行结果主要基于模拟数据测试，尚未经历 2026 年真实高考录取周期的实战检验，结果准确性有待验证。
- **覆盖范围有限**：目前部分高级功能（如定向生、艺术类等特殊批次）尚未完全自动化。

## 4. 项目介绍

本系统采用 **“单主控 + 领域专家技能 (Skills)”** 架构，运行于 Trae Solo 环境。

### 核心方法论
本项目的底层决策逻辑源自对市面上**某名师高考规划方法论的“蒸馏”与结构化处理**（非官方合作，仅为方法论复现）。我们将其转化为可由 AI 执行的 6 阶段流水线。
> **具体方法论 Notebook 与参考资料请参阅：[参考文档.md](./参考文档.md)**

### 六阶段流水线
1. **阶段 0：数据准备 (Data Pipeline)** - 将官方 PDF/Excel 招生计划自动化转化为结构化数据。
2. **阶段 1：初步筛选 (Constraint Filter)** - 基于分数、位次、选科、体检等刚性条件圈定基本盘。
3. **阶段 2：选专业 (Major Evaluator)** - 深度评估专业价值（体制内、宽度、地域、陷阱）。
4. **阶段 3：选学校 (School Matcher)** - 寻找高性价比、行业认可度高的院校。
5. **阶段 4：填报策略 (Strategy Planner)** - 基于“冲-稳-保”原则生成 96 个志愿排列方案。
6. **阶段 5：精调终审 (Special Advisor)** - 特殊群体适配与安全风险终审。
